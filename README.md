## ✨ Sobre o projeto

Este projeto simula uma Pokédex interativa, permitindo buscar Pokémon por nome ou ID, além de navegar entre eles.

O foco principal foi praticar:
- Consumo de API REST
- Manipulação do DOM
- Estruturação de projeto front-end
- Lógica com JavaScript puro

---

## 🎮 Funcionalidades

- 🔎 Busca de Pokémon por nome ou ID  
- ⬅️➡️ Navegação entre Pokémon  
- 🖼️ Exibição de imagem oficial  
- 📊 Informações básicas (nome e número)  
- 🎨 Interface inspirada na Pokédex clássica  

---

## 🛠️ Tecnologias

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- PokéAPI
